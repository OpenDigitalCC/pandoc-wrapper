# Superseded templates (archived 2026-06)

These were the hand-maintained Eisvogel forks. They are replaced by the
three-layer structure: `eisvogel-wrapper.latex` (pristine Eisvogel 3.4.0 +
`pipeline-preamble.tex`) and the standalone `mvp.latex`. All brands now select
`eisvogel-wrapper`.

Kept for reference only. A document still pinning one of these by name in its
front matter should switch to `template: eisvogel-wrapper` (or `mvp`).
